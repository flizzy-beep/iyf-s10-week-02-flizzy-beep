## Project Description
A fully responsive, styled portfolio website built as part of IYF Weekend Academy Season 10, Week 2. This project transforms a plain HTML portfolio into a beautifully styled, responsive website using CSS Flexbox, Grid, custom properties, and mobile-first design principles.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Google Fonts (Poppins + Open Sans)
- Responsive Design / Media Queries

## Features
- Responsive navigation with CSS-only hamburger menu
- Hero section with full viewport height and overlay
- CSS custom properties for typography scale and color palette
- Flexbox card row layout
- CSS Grid project gallery (1 → 2 → 3 columns responsive)
- Magazine-style layout with CSS Grid areas
- Styled contact form with focus/error states
- Multi-column responsive footer
- Hover and focus states on all interactive elements
- Mobile-first responsive design (320px → 1440px)

## Files
- `index.html` — Home page with hero, about, and card section
- `projects.html` — Projects page with grid gallery and magazine layout
- `contact.html` — Contact page with styled form and sidebar
- `styles.css` — All styles (reset, typography, colors, layouts, responsive)
- `box-model-practice.html` — Box model exercises (Task 3.2)

## Lessons Learned
- How CSS `box-sizing: border-box` prevents unexpected layout overflow
- Using CSS custom properties (variables) for consistent design systems
- The difference between Flexbox (1D) and Grid (2D) layouts
- Mobile-first design: writing base styles for mobile, then expanding upward with `@media (min-width: ...)`
- How to create accessible focus states for keyboard navigation

## Challenges Faced
- Understanding why `padding` adds to element width by default and how `box-sizing` solves it
- Getting the CSS-only hamburger menu to work without JavaScript
- Making the Grid magazine layout collapse properly on mobile
